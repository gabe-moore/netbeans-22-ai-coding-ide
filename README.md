## Overview

NetBeans is a desktop IDE created for developers who move between languages, frameworks, and project styles. It combines editing, project browsing, tooling, and workflow helpers inside a modular environment that can be shaped to fit different development setups.

Release 22.0.0 focuses on modern software work that benefits from polyglot support, plugin-based expansion, and AI-assisted productivity. It works well when you want one workspace for Java, Python, JavaScript, and other supported languages while keeping build, debug, and source control tasks close at hand.

## Configuration

NetBeans settings are usually handled through IDE preferences and project-level configuration files. Depending on your setup, you can adjust plugin selection, editor behavior, build profiles, and language tooling from the application UI.

Example configuration pattern:

```ini
editor.fontSize=14
plugins.enabled=core,java,python,javascript
ai.assistant=true
versionControl.enabled=true
```

Project-specific settings are generally saved with the workspace or inside the project directory, while global preferences stay in the user profile.
